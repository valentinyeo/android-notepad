# Claude Code Preferences

## Terminal Tab Naming
When running commands in the terminal, keep "android-notepad" at the front of the description so the tab is easily identifiable.

Example: `android-notepad: Build debug APK` instead of `Build debug APK`

## Build Commands
- Debug build: `./gradlew assembleDebug`
- Release build: `./gradlew assembleRelease`
- Clean: `./gradlew clean`

## Project Info
- Package: `com.simplenotepad`
- Min SDK: 26 (Android 8.0)
- Target SDK: 34
- Language: Kotlin
- UI: Jetpack Compose + Material 3
